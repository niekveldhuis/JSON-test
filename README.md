 # ORACC-JSON
 
This repository contains a [Jupyter Notebooks](http://jupyter-notebook-beginner-guide.readthedocs.io/en/latest/what_is_jupyter.html) that allow you to download JSON-format files from [ORACC](http://oracc.org) and parse those for use in computational projects such as Social Network Analysis or various types of Computational Text Analysis.

The Notebooks were written by Niek Veldhuis, UC Berkeley, and may freely be downloaded, used, altered and distributed.
